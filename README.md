# First-Repo
Portfolio Repository

All project files, including individual source code , project report pdf and a comprehensive ZIP archive, are attached for your review. The ZIP file contains the complete, executable application to ensure optimal functionality.


The hamburger menu is intentionally implemented for optimal user experience on smaller screen sizes (600px, 425px, and 401px viewports)

At 768px screen size, the design transitions to a fully visible navigation bar, eliminating the need for hamburger menu functionality

This decision follows modern responsive design principles where intermediate tablet sizes can accommodate traditional navigation elements
